Employment Research Data Center 

Welcome to the National Job Training Partnership Act (JTPA) Study Public Use Data CD. This document provides basic information about the files on this CD. Clicking on underlined blue text in the sections below will link you to that document.

Acrobat Reader is required for many documents on this CD. If you do not have Acrobat Reader on your system, you may install it from this CD following the instructions in the "Installing Acrobat Reader" section below.

The contents of this CD can be found in the CDcontents document. The data summary provides a brief overview of the data.

The DATA directory contains the National JTPA Study datasets and associated file contents and variable means. The datasets are organized in 13 separate data directories whose structure is explained in njs_cd_contents_032503.doc . The data were provided in SAS for Windows (.sd2 or .sd7 file) and Stata format (.dta file) by Abt Associates.

In addition to the formats of the data described above, the Upjohn Institute has formatted the data into SAS export/transport files, which can be found in the “Upjohn Institute Created Files” folder within each data directory.

All datasets have been zipped; to open a zipped file, double-click on it and then select the Extract toolbar button and save the unzipped files in the folder of your choice.

For those using the SAS export/transport files, the file Import.sas in the PROGRAMS folder contains a sample program to import the SAS export/transport file into the user’s SAS system.

The file contents and variable means calculated by the Upjohn Institute can be found in the “Upjohn Institute Created Files\Contents & Means” subdirectory within each data directory.

Additional information about the data can be found in the DOCUMENTATION directory. This information was provided by Abt Associates. This directory contains the public use data documentation, an addendum to the documentation, as well as codebooks for individual datasets. Additional documentation refers to sample design; a file outlining the structure of the datasets is also included.

In addition, the “Data\UI and SSA” and “Data\replacement_files” data subdirectories contain individual documentation files that pertain to the data in these subdirectories only.

The REPORTS directory contains the executive summary, the final report, a summary report, and the 18-month impact report.

The IMAGES directory contains the header image used in this file. This directory does not contain any information about the project itself.

Open the INSTALL folder in Windows Explorer

 

Double-click on the rp500enu.exe application

 

Follow the directions given on the screen
